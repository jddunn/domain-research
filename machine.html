    Intro

    The software is run in Processing, and uses the NeuroSky MindWave in conjunction with an Arduino Uno connected to a pulse sensor. Using BPM (Heartbeats per minute) to detect when the user's heartrate falls below resting level, the program knows the user is asleep. Using the blink detection and brain wave scanner on the MindWave, the program will know and alert the user when he is in REM sleep, and thus in a dream state. Once the user has become lucid in his dream, he will be able to send back biometric data to the program, which will translate the user's end of communication to run various functions in the program, such as activating the audio-based, real-time clock interface.
